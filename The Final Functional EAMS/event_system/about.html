About This System

Event Attendance Management System

The Event Attendance Management System is designed to simplify and automate attendance monitoring through QR code scanning, user registration, event creation, and real-time check-in/check-out. This system provides a smooth workflow for administrators, organizers, and participants.

Main Features

  • ✔ Unique QR Code for every registered user
  • ✔ Printable QR ID card
  • ✔ Mobile-optimized QR Scanner
  • ✔ Role-based dashboards (Admin, Organizer, User)
  • ✔ Attendance history with timestamps
  • ✔ Automated reporting & PDF export
  • ✔ User profile editing and secure password change
  • ✔ Email notifications for new events

Purpose

This system replaces manual attendance sheets with an efficient digital platform. It ensures accuracy, quick scanning, reduced paperwork, and professional event tracking.

Technologies Used

  • ✔ HTML, CSS, JavaScript
  • ✔ PHP (API backend)
  • ✔ MySQL Database
  • ✔ PHP QRCode Library
  • ✔ HTML5QR Scanner
  • ✔ jsPDF for report export

System Description

Users register and receive a unique QR code stored in the system. Organizers can create events, manage attendance, and view analytics. The scanner supports mobile devices for easy on-site validation. Admins can manage all users, events, and logs.
